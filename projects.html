Personal Projects

The purpose of this page is to show you some of my interests in a more comprehensive way than just stating the name of a subject. In the following section I will describe, using text, sketches and pictures, how I would like to improve the world in which we live. It is my personal belief that if a task does not have this goal then it is pointless to work on that.
All of the following projects, from simple theories to more tangible ones, are pursued in my free time. In these projects I put my passions and hope for a better future.

Necessary conditions for a project to be carried on

For each project a series of key factors are considered:

  • required skills: it has to be possible for me to realize it.
  • feasibility of the project considering today's technologies: I want to work on something viable.
  • the impact of the project: the more people it would help the more important it is, the more it would improve life on Earth (and beyond) the better.
  • cost of the components: it has to be possible to build a working prototype for under 1000 euro.

Space Suit

One of the main problems in space exploration at our point in time is the creation of an environment that allows life in space where radiations cannot enter. A goal in this direction would be the engineering of a new kind of space suit that better protects the astronauts from cosmic radiations.
